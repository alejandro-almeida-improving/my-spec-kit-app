# Quickstart: Developer Productivity Tools

## Prerequisites
- Node.js 18+
- npm or pnpm

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Add UI Components**:
    ```bash
    npx shadcn@latest add sidebar sonner card input button select textarea label tabs
    npm install date-fns lorem-ipsum
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Access the App**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## Testing

1.  **Run Tests**:
    ```bash
    npx playwright test
    ```

2.  **View Report**:
    ```bash
    npx playwright show-report
    ```
